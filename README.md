# Scraping-Facebook-pages
Scraping Facebook Hespress page

# Libraries
- Selenium
- BeautifulSoup 

# Example of scraping a Facebook Post of Hespress Page.

Output contains:

- Poste ID
- Date
- Message
- Caption
- Link
- Type of the post
- Image og the post
- Number of Likes
- Number of Shares
- Number of Comments
